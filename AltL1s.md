
# Tempo

Tempo 是由 Stripe 推动的、以稳定币作为 gas 的支付优先型区块链。

* **Mainnet 已正式上线（2026.3）**，进入真实运行阶段
* 发布 **Machine Payments Protocol（MPP）**，定义 agent payments 标准
* 引入 **session-based payments（类似 OAuth for money）**，支持授权范围内自动支付
* 支持 **micropayment aggregation（小额支付批量结算）**
* 上线 **payments directory（100+ 服务）**，形成初步支付网络
* 披露与 **OpenAI、Anthropic、Visa、Mastercard、Shopify 等**合作推进

* Monad 主网 November 24, 2025 启动以来已上线 132 个项目。
* TVL in DeFi: $340.73m (March 19, 2026)
* Monad 主网进行硬分叉，升级至 MONAD_NINE (March 19, 2026)

# IoTeX

IoTeX 正在从 DePIN 公链转型为面向 AI 的现实世界数据基础设施。

* TVL in DeFi: $2.19 m (March 19, 2026)
* 强化 **Physical AI / machine data** 叙事，将 W3bstream 与设备数据定位为 AI 输入层
* 提出 **Anti-Roadmap**，明确放弃固定路线图，转向根据 AI 与 DePIN 生态变化进行自适应开发
* February 21, 2026：ioTube bridge 发生安全事件（$2 ~ $4.3 million），随后推出100%补偿方案。
* Mainnet v2.3.4，这是一个安全补丁，永久将 29 个攻击者地址列入黑名单，并冻结了大约 4500 万个 IOTX 代币。

# Conflux

Conflux 正在成为以中国及亚洲市场为核心的合规稳定币与 RWA 结算基础设施。

* TVL in DeFi: $7.76m (March 19, 2026)
* 持续推进 Conflux 3.0, 但技术更新更多停留在方向与规划层
  - 15,000 TPS with Tree-Graph consensus protocol
  - AI Agent Integration
* 加强离岸人民币稳定币（CNH）与跨境结算叙事
* 推广 Keyless Wallet（MPC + abstraction），强调 Web2 onboarding
* 强化与 **香港 / 亚洲金融机构** 的合作预期
